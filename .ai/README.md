# .ai/

Extra orientation for AI coding tools working in this repo. The human-facing docs are the
root [README.md](../README.md) (setup + running) and [TESTING.md](../TESTING.md) (the full
test reference); this folder holds agent-oriented context that complements them without
duplicating.

- [context.md](./context.md) — what this repo tests, the CD system around it, and the
  marker contract that makes cleanup safe.
- [related-repos.md](./related-repos.md) — the sibling action repos and how they connect.
- [glossary.md](./glossary.md) — domain terms used across the workflow and docs.

Entry point is [AGENTS.md](../AGENTS.md) at the repo root, which imports the essentials from
here. When adding future AI-helper material, put it in this folder and link it from
[AGENTS.md](../AGENTS.md) or this index.
